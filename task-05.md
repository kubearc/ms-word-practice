```markdown
# 📝 Microsoft Word Task – Create a Personal Profile Document

## 🎯 Objective

To practice using various Word features to create a structured and visually appealing personal profile.

---

## 📄 Task Instructions

Create a document titled **"My Personal Profile"** using the steps below:

---

### ✅ 1. Use the *Home* Tab

- Type your **full name** at the top and make it **bold**, **underlined**, and **center aligned**.
- Below your name, write a short paragraph (4–5 lines) about yourself.
- Use **italic** formatting for your hobbies and interests.
- Use a **numbered list** to describe your top 3 strengths.

---

### ✅ 2. Use the *Insert* Tab

- Insert a **profile photo** (can be any image).
- Add a **horizontal line** below your name to separate it from the rest of the content.
- Insert a **SmartArt** diagram to show your academic journey or career path.
- Insert a **text box** to write a motivational quote you like.

---

### ✅ 3. Use the *Design* Tab

- Choose a **theme** that suits a professional profile.
- Apply a **style set** to give your document a modern look.
- Use **document formatting styles** to style the paragraph and headings.

---

### ✅ 4. Use the *Layout* Tab

- Set the **page size** to *A4*.
- Adjust the **margins** to *Moderate*.
- Add **spacing before and after paragraphs** for better readability.

---

## 📌 Final Step

Save your file as: `Personal_Profile.docx`

---
```
