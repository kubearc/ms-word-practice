# 📘 Microsoft Word Task – Using the Reference Tab

This is a beginner-friendly **Microsoft Word task** focused on using the **Reference tab** features such as **Table of Contents**, **Footnotes**, **Citations**, and **Bibliography**.

---

## 🎯 Objective

To practice using the tools in the **References** tab to add a Table of Contents, Footnotes, Citations, and a Bibliography to a Word document.

---

## 📌 Task Instructions

1. **Open Microsoft Word** and start a new blank document.

2. **Add the following headings and content**, and apply styles using the **Home tab**:

   - **Heading 1**: *Introduction*  
     - Write a short paragraph about the importance of referencing in documents.

   - **Heading 1**: *History of Microsoft Word*  
     - Add 3–4 lines about the release and evolution of MS Word.

   - **Heading 1**: *Features of MS Word*  
     - Add 2–3 bullet points about popular features like formatting, spell check, templates, etc.

3. **Insert a Table of Contents** at the top:
   - Go to the **References tab** → Click **Table of Contents** → Choose a built-in style.

4. **Add a Footnote**:
   - Select a word or sentence in the “History” section.
   - Go to **References tab** → Click **Insert Footnote**.
   - Add a note like: `Source: Microsoft Official Site`.

5. **Add a Citation**:
   - In the “Features” section, select a sentence and click **Insert Citation**.
   - Choose **Add New Source**, then enter:
     - **Type**: Book  
     - **Author**: John Smith  
     - **Title**: Mastering Microsoft Word  
     - **Year**: 2020  
     - **Publisher**: TechBooks Press

6. **Insert a Bibliography**:
   - Go to the end of the document.
   - Click **Bibliography** → Select any format (e.g., "References").

7. **Update the Table of Contents**:
   - Click on it → Choose **Update Table** → Select **Update Entire Table**.

---

## ✅ Expected Outcome

- A well-structured Word document with:
  - A **Table of Contents** based on headings
  - At least one **Footnote**
  - At least one **Citation** and a **Bibliography**
  - Professional and organized layout using the Reference tab
