# MS Word Comprehensive Practice Project
## Project Title

**IT Training Institute Annual Report 2026**

### Objective

Create a professional 8–10 page report using all major MS Word features. This task covers Home, Insert, Design, Layout, References, Mailings, Review, and View tabs.

---

# Part 1: Create the Cover Page

Create a cover page containing:

**Title:** IT Training Institute Annual Report 2026

**Prepared By:** Your Name

**Organization:** ABC IT Training Institute

**Date:** 01 June 2026

**Institute Logo:** Insert any logo/image

### Formatting Requirements

* Title Font Size: 24
* Bold and Center Aligned
* Subtitle Font Size: 16
* Add a page border
* Add a watermark "CONFIDENTIAL"
